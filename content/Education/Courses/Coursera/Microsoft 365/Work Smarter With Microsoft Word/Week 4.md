# Week 4: Working with Graphics and Visual Elements

## Adding Graphics

### Insert Symbols and Special Characters

In Word you can:
- Insert symbols and special characters
- Add images and media
- Add illustrations, icons, and screenshots
- Resize and crop
- Remove backgrounds
- Compress images

#### Inserting Symbols:
1. Go to the **Insert** tab
2. Click **Symbol**
3. Choose from various symbols (currency, degrees, etc.)

#### Special Characters:
- Found in the Special Characters tab
- Contains elements of grammar and other commonly used terms in word processing

#### Wingdings:
- Series of fonts that renders letters as symbols
- Access from the Symbol pane

---

### Add Images to a Document

An **Illustration** is a graphic file created in or inserted into a Word document.

#### Inserting Illustrations:
1. Place your cursor where you want the illustration
2. Go to the **Insert** tab
3. Choose the type of illustration you want to insert

#### Image Sources:
- **Stock Images**: Insert images from online sources
- **From Files**: Insert images from your device

#### Picture Format Tab:
- Appears automatically when you insert a photo
- Contains tools to edit the inserted image

---

### Resize an Image

#### Methods to Resize:
1. **Drag Method**:
   - Click on the image to select it
   - Drag the corner handles to resize
   - Hold Shift while dragging to maintain proportions

2. **Picture Format Tab**:
   - Select the image
   - Go to the **Picture Format** tab
   - Enter specific height and width values

#### Additional Image Editing:
- **Crop**: Trim unwanted parts of an image
  - Select the image
  - Click **Crop** in the Picture Format tab
  - Drag the crop handles
  - Press Enter when finished

- **Crop to Shape**:
  - Select the image
  - Click **Crop** dropdown
  - Choose **Crop to Shape**
  - Select desired shape

- **Compress Pictures**:
  - Reduces file size
  - Select the image
  - Click **Compress Pictures**
  - Choose desired compression options

- **Remove Background**:
  - Helps remove unwanted parts of an image
  - Select the image
  - Click **Remove Background**
  - Use the tools to mark areas to keep or remove

---

## Using Images in a Document

### Adjust Image Appearance

Word allows you to:
- Correct and recolor images
- Apply artistic effects and styles
- Integrate pictures and text
- Change a picture's position
- Add captions
- Insert and format screenshots

#### Adjusting Brightness and Contrast:
1. Select the image
2. Go to the **Picture Format** tab
3. Click **Corrections**
4. Choose from preset options or click **Picture Corrections Options** for more control
   - **Contrast**: Adjusts the tone of color in the image
   - **Brightness**: Adjusts the white in the image

#### Color Adjustments:
1. Select the image
2. Go to the **Picture Format** tab
3. Click **Color**
4. Choose from options:
   - **Color Saturation**
   - **Color Tone**
   - **Recolor**

#### Advanced Color Options:
- **More Variations**: Choose colors not offered in presets
- **Set Transparent Color**: Make a specific color transparent
- **Picture Color Options**: Opens menu with additional settings

#### Artistic Effects:
1. Select the image
2. Go to the **Picture Format** tab
3. Click **Artistic Effects**
4. Choose a preset effect
5. For more options, click **Artistic Effects Options**

#### Transparency:
- Makes the picture see-through
- Adjust level using the transparency slider
- For more options, click **Picture Transparency Options**

#### Other Image Options:
- **Change Picture**: Replace with another image
- **Reset Picture**: Remove all formatting changes
- **Picture Styles**: Apply preset styles to the image

---

### Integrate Pictures and Text

#### Text Wrapping:
1. Select the image
2. Go to the **Picture Format** tab
3. Click **Wrap Text**
4. Choose a wrapping option

> **Tip**: You can also access text wrapping options by clicking the Layout Options button that appears when you select an image

#### Custom Wrap Points:
1. Select the image
2. Click **Wrap Text**
3. Choose **Edit Wrap Points**
4. Drag the points to customize how text wraps around the image

#### Positioning Images:
1. Select the image
2. Click **Position**
3. Choose a pre-defined position
4. For more options, click **More Layout Options**

#### Rotating Images:
- Click **Rotate** command to choose preset rotations
- Or select the image and use the rotation handle (circular arrow)

#### Alignment Guides:
- Green lines appear when an image is aligned with other objects
- Enable by clicking **Align** then checking **Use Alignment Guides**

#### Adding Captions:
1. Select the image
2. Go to the **References** tab
3. Click **Insert Caption**
4. Customize caption in the dialog box
5. Click **OK**
6. Format the caption like regular text

---

### Insert and Format Screenshots

#### Taking a Screenshot:
1. Go to the **Insert** tab
2. Click **Screenshot**
3. Choose from thumbnails of open windows

#### Screen Clipping:
1. Go to the **Insert** tab
2. Click **Screenshot**
3. Select **Screen Clipping**
4. Drag to select the area you want to capture

> **Note**: Only maximized windows can be clipped.

---

## Text Boxes and Pull Quotes

### Create Text Boxes and Pull Quotes

**Text boxes** are mainly made for text but can also hold pictures and shapes.

**Pull Quotes** are lines designed to capture the reader's attention.

**Sidebar titles** are designed to sit at the margins of the document to provide additional information.

#### Word lets you:
- Insert pre-defined text boxes
- Apply different styles
- Manually format text boxes
- Add shadow and 3D effects
- Change the direction of text
- Save customized text boxes

#### Inserting Text Boxes:
1. Go to the **Insert** tab
2. Click **Text Box**
3. Choose from the available options
4. For sidebars, select from the same menu

---

### Format Text Boxes and Pull Quotes

When you add a quote or sidebar, a special tab appears with formatting options:

#### Formatting Options:
- Choose a style from the Styles gallery
- Edit using commands next to the Styles library
- Change text direction using the **Text Direction** command

#### Saving Custom Text Boxes:
1. Format a text box as desired
2. Click the **Text Box** command
3. Choose **Save Selection to Text Box Gallery**
4. Name your custom text box
5. Click **OK**

---

## Using Shapes in Document

### Draw Shapes

In Word, you can:
- Draw, resize, and style shapes
- Insert a drawing canvas
- Set text wrapping options
- Configure wrap points
- Add text to shapes
- Add captions

#### Adding Shapes:
1. Go to the **Insert** tab
2. Click **Shapes**
3. Choose the shape you want
4. Click once for default size or click and drag to set custom size

#### Shape Formatting:
- Use the **Shape Format** tab to change appearance
- Apply the same formatting options as text boxes

#### Drawing Canvas:
- Acts as a container for shapes
- Access by clicking **Shapes** then **New Drawing Canvas**

---

### Modify Shapes

Modifying shapes and wrap points works the same way as with text boxes.

---

## Word Art and Complex Illustration

### Add WordArt and Other Text Effects

In Word you can:
- Insert WordArt and SmartArt
- Format text with Drop Caps
- Add text and images to SmartArt
- Use various types of SmartArt

#### Using WordArt:
1. Go to the **Insert** tab
2. Click **WordArt**
3. Choose a style
4. Type your text
5. Use the **Shape Format** tab to change font or color

#### Adding Drop Caps:
A Drop Cap is an enlarged capital letter at the beginning of a paragraph.

1. Place cursor in paragraph
2. Go to the **Insert** tab
3. Click **Drop Cap**
4. Choose a style

---

### Create Complex Illustrations with SmartArt

#### Inserting SmartArt:
1. Go to the **Insert** tab
2. Click **SmartArt**
3. Choose the type that suits your needs
4. Click **OK**

#### Customizing SmartArt:
- Use the **SmartArt Design** tab for customization
- Click **Add Shape** to add more elements
- Use **Text Pane** to type text directly instead of in the SmartArt
- Use **Promote** or **Demote** to change hierarchy
- Use **Format** tab to change appearance
