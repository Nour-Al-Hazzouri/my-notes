- Document object is a built-in object that has many properties and methods that can be used to manipulate the structure and the style and project
- Document object is a child of the browser object which is the top node that represents a tab in the browser
- This DOM has a parent-child-sibling relationship, the parent is the main object, the child object is the sub-object coming from the parent object, siblings are objects on the same level from the same parent
- Elements attributes are considered nodes in this hierarchy and do not participate in this parent-child-sibling relationship
