# Week 3: Formatting Text and Working with Tables

## Formatting Text

### Apply Character Formatting

**Microsoft Word and fonts**: A font is a complete set of characters used to create text in a Microsoft Word Document.

#### Formatting Methods in Microsoft Word:
- Fonts
- The Fonts Group Command in the Home tab
- Highlighting Text
- The Format Painter
- The MiniTool Bar

You can change different elements in a font including:
- Size
- Type
- Color
- Spacing
- Effects and other options

#### How to Change Font:
1. Select the text you want to format
2. Use the mini tool bar that pops up
3. Choose your desired formatting options

#### Using Format Painter:
1. Select text with the formatting you want to copy
2. Click **Format Painter** from the Clipboard group
3. Select the text where you want to apply the formatting

> **Note**: Format Painter settings aren't saved in clipboard, so only one style can be used at a time.

---

### Align Text Using Tabs

Tabs and tab stops control where your text is horizontally aligned and can be customized using the ruler.

#### Displaying the Ruler:
1. Go to the **View** tab
2. Check the **Ruler** box

#### Editing Tab Settings:
1. Select your text
2. Go to the **Home** tab
3. In the Paragraph section, click **Settings**
4. Choose **Tabs** from the bottom of the window
5. Adjust tab settings as needed

---

### Display Text as List Items

#### Types of Lists in Word:
- Bulleted lists
- Numbered lists
- Multilevel lists

#### Creating a List:
1. Select text (if already typed)
2. Choose the type of list from the Paragraph group
   - OR -
1. Click the list type button
2. Start typing your list
3. Press Enter twice when finished to exit list mode

#### Customizing Lists:
- Click **Define New Bullet** to customize bullet style
- This option exists for all list types

---

### Control Paragraph Layout

In Microsoft Word you can:
- Control paragraph layout
- Apply borders and shading
- Apply styles to text
- Manage formatting

#### Paragraph Formatting Options:
- **Home tab**: Contains paragraph alignment options and indentation commands/markers
- **Layout tab**: Contains:
  - Margin tool to change page margins
  - Line and paragraph spacing options
  - Hyphenation options

#### Automatic Hyphenation:
Word can automatically hyphenate your document. For words you don't want hyphenated:
- Enter a non-breaking hyphen: **Ctrl + Shift + Underscore**
- Enter a non-breaking space: **Ctrl + Shift + Space**

---

### Apply Borders and Shading

- **Borders**: Lines that wrap around text
- **Shading**: Shadow of color applied to selected text within a page

#### Applying Borders:
1. Select the text
2. In the **Home** tab, from the Paragraph section, choose the border style you want
3. For advanced options, open the dialog box to choose styles and colors

#### Applying Shading:
1. From the same section, choose the color you want
2. For more options, open the dialog box and select the Shading tab

---

### Apply Styles

#### Accessing Styles:
- Choose styles from the **Home** tab
- Go to **Design** tab for more style options
- From the Home tab, open the Styles dialog for additional options

#### Style Inspector Tool:
Shows the style and formatting applied to currently selected text

**Keyboard Shortcut**: Press **Ctrl + Shift + S** to open a small window to choose a style to apply to selected text

---

### Manage Formatting

#### Revealing Formatting:
1. Press 'Collapse' on Styles in Home tab
2. Click Style Inspector command
3. Click the Reveal Formatting command
4. Close other task panes if desired

#### Formatting Task Pane Features:
- Top section shows selected text (or examines area around cursor if none selected)
- Body shows formatting elements separated into 3 levels:
  - Font
  - Paragraph
  - Section
- Click blue hyperlinks to open dialog boxes for specific settings
- Two checkboxes at bottom customize the view

#### Changing Font Using Replace:
1. Press **Ctrl + H**
2. Click **More**
3. Ensure cursor is in "Find what" field
4. Click **Format**
5. Select **Font**
6. Choose font to find
7. Repeat for "Replace with" field
8. Click **Replace** or **Replace All**

---

## Adding Tables

### Insert a Table

#### Table Components:
- **Rows**: Sections of data organized horizontally
- **Columns**: Sections of data organized vertically
- **Cells**: Boxes that contain the actual data

#### Methods to Insert Tables:

**Method 1: Table Grid**
1. Go to **Insert** tab
2. Click **Table**
3. Drag across the grid to select rows and columns

**Method 2: Insert Table Dialog**
1. Go to **Insert** tab
2. Click **Table**
3. Select **Insert Table**
4. Input the number of rows and columns desired
5. Click **OK**

**Method 3: Draw Table**
1. Go to **Insert** tab
2. Click **Table**
3. Select **Draw Table**
4. Draw your table (recommended for graphics tablet users)

#### Table Tools:
After inserting a table, two new tabs appear:
- **Table Design**: Changes the appearance of the table
- **Layout**: Changes the structure of the table

#### Additional Table Options:
- **Quick Tables**: Pre-designed tables for simple tasks
- **Excel Spreadsheet**: Inserts an Excel window for data entry (press Esc when finished)
- **View Borders**: Toggle table borders on/off in the Layout tab

---

### Modify a Table

In Word, you can:
- Edit rows and columns
- Customize cell margins
- Set table titles

#### Selecting an Entire Table:
1. Place cursor in the table
2. Click the 4-headed arrows icon that appears

#### Editing Table Elements:
- The **Layout** tab contains tools organized by table part
- To edit cell properties:
  1. Go to Layout tab
  2. Click **Properties**
  3. Adjust width, alignment, indent, text wrapping, etc.

#### Adding Rows or Columns:
1. Place cursor where you want to add
2. Click the **+** button that appears

#### Deleting Rows or Columns:
1. Select the rows/columns to delete
2. Right-click and select **Delete**
   - OR -
1. Select the rows/columns
2. Go to Layout tab
3. Click **Delete** and choose appropriate option

#### Adjusting Size:
- Double-click a cell border to fit content
- Go to Layout tab and set specific dimensions

#### Cell Margins:
- Adjust white space between table and text
- These settings apply to the whole table

#### Adding a Title:
1. Go to **Properties**
2. Select **Alt Text** tab
3. Add title (not visible but accessible by screen readers)

---

### Format a Table

Word provides many formatting options:
- Applying table styles
- Changing fonts
- Altering table structure

#### Applying Table Styles:
1. Select the table
2. Go to **Table Design** tab
3. Choose a style from the gallery

#### Changing Fonts:
- Use Home tab like normal text
- Modify table style to change font for all tables with same style:
  1. Right-click the style
  2. Choose **Modify Table Style**
  3. Make desired changes

#### Additional Formatting Options:
- **Table Style Options**: Toggle features on/off
- **Borders**: Customize border appearance

---

### Convert Text to Table

#### Converting Text to Table:
1. Ensure data is correctly separated (e.g., comma then space)
2. Select the text
3. Go to **Insert** tab
4. Click **Table**
5. Select **Convert Text to Table**
6. Adjust options in the dialog
7. Click **OK**

Example of properly formatted text for conversion:
```
Hi, bye, lol
Hello, goodbye, haha
Greetings, farewell, laughs
```

#### Converting Table to Text:
1. Select the table
2. Go to **Layout** tab
3. Click **Convert to Text**
4. Choose separator options
5. Click **OK**

## Working with Tables and Cell Layout

### Sort Table Data

1. Select the column to sort
2. Go to **Layout** tab
3. Click **Sort**
4. Choose desired options
5. Click **OK**

---

### Control Cell Layout

#### Changing Text Direction:
1. Select the text
2. Go to **Layout** tab
3. Click **Text Direction**
4. Choose desired orientation

#### Merging Cells:
1. Select cells to merge
2. Go to **Layout** tab
3. Click **Merge Cells**
4. Choose appropriate option
5. Click **OK**

---

## Managing Lists

### Sort a List

#### Types of Sortable Data:
- Text
- Numbers
- Dates

#### Sorting a List:
1. Select the list
2. From the **Home** tab, click **Sort**
3. Choose desired options in the dialog
4. Click **OK**

> **Note**: Tables can also be sorted using the same method.

---

### Renumber a List

1. Place cursor at the end of the last line in the list
2. Right-click
3. Choose:
   - **Continue Numbering**
   - **Restart at 1**
   - **Set Numbering Value** (opens dialog to set specific value)

---

### Customize a List

#### Creating a Multilevel List from Existing Text:
1. Select the text
2. Click **Multilevel List** command from Home tab
3. Choose desired style

#### Creating Sublevels:
1. Select specific text in the list
2. Click **Increase Indent** command
3. The text becomes a sub-list

#### Creating a Multilevel List from Scratch:
1. Click the Multilevel List command
2. Begin typing
3. Press Tab to create sublevel
4. Press Shift+Tab to return to higher level

#### Customizing Lists:
1. Click the Multilevel List command
2. Select **Define a New Multilevel List**
3. Adjust settings as desired
4. Click **OK**

#### Creating a New List Style:
1. Click the Multilevel List command
2. Select **Define New List Style**
3. Customize style options
4. Click **OK**
