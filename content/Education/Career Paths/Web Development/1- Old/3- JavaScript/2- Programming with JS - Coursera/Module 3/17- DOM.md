- DOM tree structure:
    
    HTML file is mapped out like a tree structure with nested objects for different parts of the page
    
- DOM:
    - A browser builds DOM, or model, for every webpage it downloads as a JS object
    - When the browser access a webpage, it download it and save it in its memory, then it build a DOM after seeing all the HTML, as a developer you can use JS to manipulate the DOM
- Commands:
    - Create an element: const variable= document.createElement(’element’)
    - Change element content: element.innerText= “Content”
    - Change element attribute: element.setAttribute(’attribute’, ‘Content’)
    - Add element: document.element.appendChild(element)
