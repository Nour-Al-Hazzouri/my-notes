# Unit 5: Place Value through 1,000,000

## Overview
This unit explores the concept of place value in our number system, extending students' understanding to numbers up to 1,000,000. Place value is a fundamental concept in mathematics that helps students understand how our number system works and provides the foundation for operations with multi-digit numbers. By mastering place value through 1,000,000, students develop the ability to read, write, compare, and manipulate larger numbers with confidence.

## Key Concepts

### Concept 1: Place Value Blocks and Models
**Definition**: Place value blocks are physical or visual representations that show the relative value of digits in different positions, with each position representing a power of 10.

**Clarification**: In our base-10 number system, each position has a value 10 times greater than the position to its right. Place value blocks help students visualize this relationship by using different-sized blocks to represent ones, tens, hundreds, and thousands. This concrete representation helps students understand the abstract concept of place value.

**Example**: The number 3,542 can be represented with 3 thousand blocks, 5 hundred blocks, 4 ten blocks, and 2 one blocks. This visual model helps students see that the digit 5 represents 5 hundreds (500), not just 5.

### Concept 2: Place Value Tables
**Definition**: A place value table organizes digits according to their position values, with columns labeled for each place (ones, tens, hundreds, etc.).

**Clarification**: Place value tables help students understand the value of each digit based on its position. They provide a structured way to read, write, and compare numbers by aligning digits with their corresponding place values. This organization reinforces the understanding that the same digit has different values depending on its position.

**Example**: In a place value table, the number 425,731 would be organized as:

```
    Place Value Table for 425,731
    
    ┌─────────────────┬─────────────┬─────────┬─────────┬──────┬──────┐
    │Hundred-thousands│Ten-thousands│Thousands│Hundreds │ Tens │ Ones │
    ├─────────────────┼─────────────┼─────────┼─────────┼──────┼──────┤
    │        4        │      2      │    5    │    7    │  3   │  1   │
    └─────────────────┴─────────────┴─────────┴─────────┴──────┴──────┘
    
    4 hundred-thousands = 400,000
    2 ten-thousands     =  20,000
    5 thousands         =   5,000
    7 hundreds          =     700
    3 tens              =      30
    1 one               =       1
                          -------
                          425,731
```

### Concept 3: Number Forms (Standard, Word, and Expanded)
**Definition**: Numbers can be represented in different forms: standard form (using digits), word form (using words), and expanded form (showing the value of each digit).

**Clarification**: These different representations help students understand the meaning behind the digits in a number. Standard form is the most compact, word form connects numbers to language, and expanded form explicitly shows the value of each digit based on its place.

**Example**: The number 347,215 can be written as:

```
    Different Forms of 347,215
    
    Standard Form:
    347,215
    
    Word Form:
    Three hundred forty-seven thousand, two hundred fifteen
    
    Expanded Form:
    300,000 + 40,000 + 7,000 + 200 + 10 + 5
    
    Expanded Form with Place Value:
    (3 × 100,000) + (4 × 10,000) + (7 × 1,000) + (2 × 100) + (1 × 10) + (5 × 1)
```

### Concept 4: Regrouping Whole Numbers
**Definition**: Regrouping involves exchanging between different place values based on the base-10 system, where 10 units of one place value equal 1 unit of the next higher place value.

**Clarification**: Understanding regrouping is essential for addition and subtraction with multi-digit numbers. It involves recognizing when to convert between place values (e.g., 10 ones = 1 ten, 10 tens = 1 hundred) and is a key skill for mental math and computational fluency.

**Example**: When representing 3,542, we can regroup it as 3 thousands, 5 hundreds, 3 tens, and 12 ones, or as 3 thousands, 4 hundreds, 14 tens, and 2 ones. Both are valid regroupings that equal the same total value.

```
    Regrouping the Number 3,542
    
    Standard grouping:
    3 thousands + 5 hundreds + 4 tens + 2 ones
    
    Regrouping some tens as ones:
    3 thousands + 5 hundreds + 3 tens + 12 ones
    
    Regrouping some hundreds as tens:
    3 thousands + 4 hundreds + 14 tens + 2 ones
    
    Regrouping some thousands as hundreds:
    2 thousands + 15 hundreds + 4 tens + 2 ones
    
    All regroupings equal the same value: 3,542
```

### Concept 5: Identifying the Value of a Digit
**Definition**: The value of a digit in a number is determined by multiplying the digit by its place value.

**Clarification**: This concept helps students understand that the same digit has different values depending on its position in a number. It reinforces the multiplicative nature of our place value system and helps students decompose numbers meaningfully.

**Example**: In the number 45,678:

```
    Value of Each Digit in 45,678
    
    ┌─────┬─────┬─────┬─────┬─────┐
    │  4  │  5  │  6  │  7  │  8  │
    └─────┴─────┴─────┴─────┴─────┘
       ↓     ↓     ↓     ↓     ↓
      ×10⁴  ×10³  ×10²  ×10¹  ×10⁰
       ↓     ↓     ↓     ↓     ↓
    40,000 5,000  600   70    8
    
    The digit 4 has a value of 40,000 (4 × 10,000)
    The digit 5 has a value of 5,000 (5 × 1,000)
    The digit 6 has a value of 600 (6 × 100)
    The digit 7 has a value of 70 (7 × 10)
    The digit 8 has a value of 8 (8 × 1)
```

### Concept 6: Multiplying and Dividing by 10
**Definition**: When a number is multiplied by 10, each digit shifts one place to the left; when divided by 10, each digit shifts one place to the right.

**Clarification**: This pattern is a direct result of our base-10 number system. Understanding these shifts helps students develop mental math strategies and prepares them for work with decimals. It also reinforces the relationship between adjacent place values.

**Example**: 

```
    Multiplying and Dividing by 10
    
    Multiplying 456 by 10:
    
    456 × 10 = 4,560
    
    ┌───┬───┬───┐      ┌───┬───┬───┬───┐
    │ 4 │ 5 │ 6 │  →   │ 4 │ 5 │ 6 │ 0 │
    └───┴───┴───┘      └───┴───┴───┴───┘
    
    Each digit shifts one place to the left,
    and a zero is placed in the ones position.
    
    Dividing 4,560 by 10:
    
    4,560 ÷ 10 = 456
    
    ┌───┬───┬───┬───┐      ┌───┬───┬───┐
    │ 4 │ 5 │ 6 │ 0 │  →   │ 4 │ 5 │ 6 │
    └───┴───┴───┴───┘      └───┴───┴───┘
    
    Each digit shifts one place to the right,
    and the last digit (0) is removed.
```

### Concept 7: Place Value Patterns
**Definition**: Place value follows predictable patterns that extend infinitely in both directions (larger and smaller).

**Clarification**: Understanding these patterns helps students work with numbers of any size. Each place value is 10 times the value of the place to its right and 1/10 the value of the place to its left. This pattern continues through larger numbers (millions, billions, etc.) and smaller numbers (tenths, hundredths, etc.).

**Example**: The pattern of place values: ones, tens, hundreds, thousands, ten thousands, hundred thousands, millions... Each place is 10 times the previous place.

```
    Place Value Patterns
    
    ×10        ×10        ×10        ×10        ×10        ×10
    ←────      ←────      ←────      ←────      ←────      ←────
    
    Millions  Hundred-   Ten-      Thousands  Hundreds   Tens      Ones
              thousands  thousands
    
    1,000,000  100,000    10,000     1,000      100       10        1
    
    ────→      ────→      ────→      ────→      ────→      ────→
    ÷10        ÷10        ÷10        ÷10        ÷10        ÷10
    
    Each place value is 10 times the value to its right
    Each place value is 1/10 the value to its left
```

## Summary
Unit 5 develops students' understanding of place value through 1,000,000, building on their knowledge of smaller numbers. Students learn to use place value blocks and tables to represent numbers, write numbers in different forms (standard, word, and expanded), regroup numbers in various ways, identify the value of digits based on their positions, and recognize patterns when multiplying or dividing by 10. These concepts provide the foundation for working with large numbers and performing operations with multi-digit numbers.

## Connection to Next Unit
The place value concepts learned in Unit 5 directly support Unit 6, which focuses on addition and subtraction through 1,000,000. Understanding place value is essential for performing these operations with large numbers, especially when regrouping is required. The ability to decompose numbers based on place value and recognize the value of each digit will help students develop efficient strategies for addition and subtraction with multi-digit numbers. Additionally, the patterns observed when multiplying and dividing by 10 will be useful in future units on multiplication and division.
