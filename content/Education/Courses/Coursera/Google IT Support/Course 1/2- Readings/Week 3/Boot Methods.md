# Boot Methods

## Introduction
- **Standard Boot**: Normal process initiated by pressing the power button
- **Alternative Methods**: Various other ways to boot a computer beyond the standard method

## Internal Boot Methods

### Partitioning
- **Definition**: Creating logical divisions on a computer's drive
- **Function**: Only one partition runs the boot process
- **Common Use Case**: Having two separate operating systems on one computer

### Dual Booting
- **Definition**: Having two operating systems on one computer
- **Function**: User chooses which system will run the boot process
- **Benefit**: Provides troubleshooting capability when one system fails

## External Boot Tools

### Device Types
- **USB Drive**: 
  - Boot resources loaded on USB storage device
  - Inserted into USB port and selected at startup

- **Optical Media**: 
  - Boot resources loaded on disk (DVD, CD, or Blu-ray)
  - Accessed through computer's optical drive

- **Solid State Boot Drive**: 
  - Boot device with no moving parts
  - Can be internal or external (like flash drive)

- **External Hot-Swappable Drive**: 
  - External hard drive for booting
  - Can be moved between computers without powering off

- **Network Boot**: 
  - Boots directly from local area network (LAN)
  - No local storage device required
  - Requires LAN connection

- **Internet-Based Boot**: 
  - Boots from secure internet source
  - Requires internet connection

## OS-Specific Boot Procedures

### Windows/Linux
- Press F2/F12/Del keys at startup to enter BIOS
- Change boot order to prioritize desired external tool
- Save changes and restart

### macOS
- Press and hold Option key at startup
- Startup Manager appears and scans for bootable devices
- Select desired bootable device from options
