# Projectors

## Overview
- **Purpose**: Display devices for sharing information with groups in the same location
- **Functionality**: Operate similarly to other computer displays
- **Common Issues**: Dead pixels, stuck pixels, image burn-in (similar to other displays)

## Connection Methods

### Display Cables
- **VGA**: Older analog connection standard
- **DVI**: Digital Visual Interface
- **HDMI**: High-Definition Multimedia Interface
- **DisplayPort**: Modern digital display interface

### Display Configuration
- **Extended Mode**: Computer desktop expanded across projector and main display
- **Mirrored Mode**: Same content shown on both projector and main display

### Connection Troubleshooting
- **Common Issues**: Cable wear, connector damage from frequent connections/disconnections
- **Symptoms**: Display flickering, signal loss
- **Best Practice**: Check cables and connectors early in troubleshooting process

## Technical Considerations

### Device Drivers
- **Resolution Issues**: Without proper drivers, projectors may default to low-resolution VGA mode (640x480 or 1024x768)
- **Solution**: Install appropriate device drivers from manufacturer's support website

### Lighting Technology
- **Incandescent Lamps**:
  - Traditional technology
  - Expensive to replace
  - Generate significant heat
  - Shorter lifespan
  - May cause projector shutdown if overheating occurs

- **LED Lights**:
  - Modern technology
  - Fewer overheating issues
  - Significantly longer lifespan
  - More energy efficient

### Calibration Requirements
- **When Needed**: After installation, reset, or relocation
- **Adjustment Types**:
  - Focus adjustment for image clarity
  - Keystone correction for image distortion
  - Geometry adjustments for square, aligned projection
- **Process**: Varies by manufacturer; refer to vendor documentation
