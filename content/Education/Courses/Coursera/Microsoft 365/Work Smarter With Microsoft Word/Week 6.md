# Week 6: Collaboration and Document Review

## Modify User Information and Share a Document

### Modify User Information

#### Viewing Document Properties:
1. Click the **File** tab
2. Go to **Info**
3. Review the document properties (some are editable)

#### Changing Username and Initials:
1. Go to **File** > **Options**
2. In the **General** tab, find the "Personalize your copy of Microsoft Office" section
3. Update your username and initials as needed

---

### Share a Document

> **Note**: Documents can only be shared when saved on a cloud storage service such as OneDrive.

To share a document:
1. Save the document to OneDrive
2. Click the **Share** button in the upper right corner
3. Enter email addresses of people you want to share with
4. Set permission levels
5. Add a message (optional)
6. Click **Send**

---

## Work with Comments and Compare Changes

### Work with Comments

#### Adding a Comment:
1. Select the text you want to comment on
2. Go to the **Review** tab
3. Click **New Comment**
4. Type your comment

#### Managing Comments:
- **Reply to a comment**: Click in the comment and type your reply
- **Resolve a comment**: Click **Resolve** (the comment will gray out)
- **Delete a comment**: Select the comment and click **Delete**

#### Navigating Comments:
- Click **Show Comments** to display comments in the document
- Use **Next** and **Previous** buttons to move between comments
  > **Note**: When these buttons are grayed out, there are no more comments in that direction

---

### Compare Document Changes

To compare two versions of a document:
1. Go to the **Review** tab
2. Click **Compare**
3. In the dialog box:
   - Select the **Original document** (earlier version)
   - Select the **Revised document** (later version)
4. Click **More** for additional comparison settings
5. Choose where to show changes (original, revised, or new document)
6. Click **OK**

After comparison, you can:
- Use the **Accept** dropdown to choose how to accept changes
- Use the **Reject** dropdown to choose how to reject changes

---

## Review a Document and Compare Changes

### Review a Document

#### Enabling Track Changes:
- Go to the **Review** tab and click **Track Changes**
- Or use the keyboard shortcut **Ctrl + Shift + E**

#### Understanding Tracked Changes:
- Deleted text appears with a strikethrough
- Inserted text appears with an underline
- Text with comments has highlighting

#### Viewing Options:
- Use the **Display for Review** dropdown to choose how to view the document:
  - **Simple Markup**: Shows the final document with indicators where changes exist
  - **All Markup**: Shows all changes inline
  - **No Markup**: Shows the final document without indicators
  - **Original**: Shows the original document without changes

- Use the **Show Markup** dropdown to select which types of changes to display

#### Managing Changes:
- Use the **Accept** button to accept changes
- Use the **Reject** button to reject changes
- Use **Next** and **Previous** buttons to navigate between changes

---

### Merge Document Changes

To merge changes from multiple reviewers:
1. From the **Review** tab, click **Compare**
2. Select **Combine**
3. Choose the original document and a revised version
4. Accept the changes you want to keep
5. Save the document

#### Reviewing Individual Changes:
1. Go to the **Review** tab
2. Click **Reviewing Pane**
3. Review each change in the pane that appears
4. Accept or reject changes as appropriate
