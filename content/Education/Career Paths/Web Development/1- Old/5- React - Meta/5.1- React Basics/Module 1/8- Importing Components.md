- The JS modules feature of standalone re-usable code is presented in React
- Each component can have its own file and be imported/exported
- Exports:
	- Default:
		Function name is the same as the file name
	- Named:
		Function name different from the file name
- Components & Module:
	- A component is a small piece of functionality
	- A module is a series of components
	- This technique of splitting your code into several modules is known as modular programming
	- React doesn't have strict rules on how you put files into folders, however, there are a few common approaches you may want to consider. One approach is to place all components in a folder named components.
	- When used, Each components will be called and have it's contents returned to the root components of our application, which is app.js.
