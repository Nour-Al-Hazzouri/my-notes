# My Workspace

My digital garden / personal notes website — built with [Quartz v4](https://quartz.jzhao.xyz/).

## What Is This?

A public version of the Obsidian vault I use for my own learning. If you're reading this on GitHub, you can visit the actual website where it's readable and navigable the way it's meant to be.

I take notes on whatever I'm learning. Mostly IT and web development, but also Islamic studies, or whatever random rabbit hole grabs my attention.
There's no timeline, sometimes I might update frequently, sometimes you might think I'm dead in my cave.

## Who's It For?

Honestly? Me. But since they exist anyway and might help someone, here they are.

If you're a beginner in web development or IT, or you just have the memory of a goldfish and need things written down — this might be useful for you. I was that person, and I still kind of am.

## Tech Stack

- **[Quartz v4](https://quartz.jzhao.xyz/)** — static site generator for Obsidian notes
- **[Obsidian](https://obsidian.md/)** — where the actual notes live
- Custom SCSS for RTL (Arabic) language support

## A Note on the Notes

Most of these were written before AI was a thing. Then, when Manus came up, I refactored most of my notes to be better organized (I recon it was Claude 3.5 or 3.7, something like that). They're not polished documentation — they're notes. Treat them accordingly.

If they help you, great. If not, Google probably has a better answer anyway :)
