# Week 2: Getting Started with Word

## Identify the Components of the Word Interface

In general, the Word interface includes the search bar, title bar, and profile picture, along with these specific components:

| Component | Description |
|-----------|-------------|
| **Quick Access Toolbar** | Located on top left; contains shortcuts to most used commands and can be customized |
| **Ribbon Interface** | Contains command groups; some commands appear gray when unavailable |
| **Contextual Tabs** | Special tabs that appear in certain circumstances (e.g., Picture Format appears when a picture is inserted) |
| **More Options Button** | Shows more specific options relating to the same commands of the group |
| **File Tab** | Opens Backstage view for saving files, creating new ones, and more |
| **Task Panes** | Display additional commands and options |
| **Dialog Box Launcher** | Small buttons that open dialogs with more options (e.g., Clipboard dialog) |
| **Galleries** | Display design options in visual format |

## Create a Word Document

### Key Features

- **Word Wrap**: Automatically wraps text to the next line
- **AutoCorrect**: Fixes common typing errors
- **Spelling and Grammar Check**: Identifies potential errors
  - Red underlines indicate grammatical mistakes
  - Blue underlines indicate simple mistakes

### Formatting Marks

Formatting marks are hidden by default but can be helpful to:
- See where lines end
- Remove unwanted spaces or errors

**How to Show/Hide Formatting Marks:**
- Press **Ctrl+Shift+8**, or
- Click the ¶ button in the Paragraph group on the Home tab

### Sharing Documents

If your document is saved in online storage like OneDrive:
1. Look for the **Share** button on the right side of the ribbon
2. Click it to open a prompt
3. Choose who to share the document with

### Printing Documents

1. Click the **File** tab
2. Select **Print**
3. Adjust printing options as needed
4. Click **Print** to complete the process

> **Note**: Print and preview are combined in one location for easier editing

## Create a Document Using a Template

Templates help you retain settings you commonly adjust (margins, tabs) and store frequently used items (text, dates, names).

### Types of Word Templates

1. **Document Templates**: Focus on the form and layout of the document
2. **Normal.dotm**: Used to remember settings such as font size and program options

### Accessing Templates

- Templates stored on your computer are shown under the **Personal** tab
- Template directories:
  - Document templates: `This PC > C > Users > [username] > Documents > Custom Office Templates`
  - MS Word templates: `AppData > Roaming > Microsoft`

## Use Document Views and Help

### Help Features

There are multiple ways to get help in Word:

1. **Search Feature**: The quickest way to get help
   - Type what you need help with in the search box

2. **Help Tab**: Access help options from the ribbon

3. **Help Task Pane**: 
   - Open using the F1 shortcut key
   - Contains searchable help topics

### Document Views

#### Protected View
- Documents downloaded from the internet open in Protected View
- To adjust these settings:
  1. Go to **Options**
  2. Select **Trust Center**
  3. Click **Trust Center Settings**
  4. Modify Protected View settings

#### View Options

The **View** tab offers many ways to view your document:

- **Page Movement**: Switch scrolling from vertical to horizontal
- **Zoom**: Increase or decrease document size
- **Multiple Document Views**: Compare documents side by side

**How to Compare Parts of the Same Document:**
1. Click **Split** in the View tab
2. The document will be divided into two panes
3. Any change in one pane will appear in the other

**How to Open Multiple Windows:**
1. Click **New Window** in the View tab
2. A second window of the same document will open
3. Changes in one window will appear in the other

## Editing a Document

### Navigate and Select Text

#### Scrolling
- Use scroll bars to move through documents
- Right-click scroll bar for additional options

#### Keyboard Navigation

| Key Combination | Action |
|-----------------|--------|
| **Page Up** | Goes up one page |
| **Page Down** | Goes down one page |
| **Home** | Goes to beginning of line |
| **End** | Goes to end of line |
| **Ctrl+Home** | Goes to beginning of document |
| **Ctrl+End** | Goes to end of document |

#### Text Selection

Selected text will be affected by commands. Selection can be done by mouse or keyboard:

| Key Combination | Action |
|-----------------|--------|
| **Shift+Right Arrow** | Select text to the right |
| **Shift+Left Arrow** | Select text to the left |
| **Shift+Up Arrow** | Select text upward |
| **Shift+Down Arrow** | Select text downward |

---

### Modify Text

#### Text Movement Options

**Drag and Drop:**
1. Select text
2. Click and hold on the selection
3. Drag to new location
4. Release to drop

**Cut, Copy and Paste:**
- Available through right-click menu or keyboard shortcuts:
  - **Ctrl+X**: Cut
  - **Ctrl+C**: Copy
  - **Ctrl+V**: Paste

#### Clipboard Features

- Stores up to 24 items at a time
- Paste options include:
  - **Keep Source Formatting**: Maintains original text format
  - **Merge Formatting**: Matches document's format
  - **Keep Text Only**: Uses default font with no formatting
  - **Paste Special**: Advanced pasting operations
  - **Set Default Paste**: Controls future paste operations

> **Tip**: You can preview pasted text before pasting by hovering over the paste option

#### Undo and Redo

- **Undo Command** (Ctrl+Z): Reverses previous actions
- **Redo Command** (Ctrl+Y): Restores undone actions

---

### Find and Replace Text

#### Navigation Pane

- Not visible by default
- Enable by:
  - Clicking **Find** in the Home tab, or
  - Using **Ctrl+F**
- Includes advanced options for finding graphics and other elements

#### Find and Replace Dialog Box

1. In the Home tab, click **Find**
2. Select **Advanced Find**

**How to Find Text:**
1. Choose **Find** tab
2. Enter what you want to find
3. Click **Find Next**

**How to Replace Text:**
1. Choose **Replace** tab
2. Enter what you want to find
3. Enter what you want to replace it with
4. Click **Replace** or **Replace All**

**Go To Feature:**
- Navigate to a specific place in the document
- Enter page number or use references
- Access through the **Go To** tab in the dialog

**More Options:**
- Click **More** for advanced search parameters
- Refine your search with options like match case or whole words only

## Proofing a Document

### Check Spelling and Grammar

- By default, Word marks potential errors as you type:
  - Grammatical errors: Blue underline
  - Spelling errors: Red underline

- Errors are detected by comparing words to Word's dictionary
- You can add custom words to the custom dictionary:
  1. Go to **Options**
  2. Select **Proofing**
  3. Click **Custom Dictionaries**

**Word Count:**
- Word automatically shows word count in the status bar
- Select specific text to see its word count

**Ignoring "Errors":**
- If a word is marked as an error but is correct:
  1. Right-click the word
  2. Select **Ignore All**

---

### Other Proofing Tools

#### Thesaurus and Translation
- **Thesaurus**: Press **Shift+F7** to find synonyms
- **Translation**: Click **Translate** in the Review tab

#### Research Feature
- Located in the References tab under Research group
- Performs wide searches about chosen words or phrases
- Can be customized:
  1. Go to **Options**
  2. Select **Trust Center**
  3. Click **Trust Center Options**
  4. Select **Research Options**
  5. Add or remove services

**Quick Access to Synonyms:**
1. Right-click a word
2. Select **Synonyms** or **Thesaurus**
3. Choose a replacement or select **Thesaurus** for more options

#### Adding Research Results to Documents
1. Find the research result you want to use
2. Click the **+** button
3. If prompted to create a bibliography, click **Create**
