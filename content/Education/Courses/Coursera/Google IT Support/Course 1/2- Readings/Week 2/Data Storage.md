# Data Storage

## Introduction
- **Purpose**: Understanding data storage measurements and terminology
- **Relevance**: Storage capacity increases with technology evolution, enabling larger file sizes

## Nomenclature Systems

### Decimal Nomenclature
- **Purpose**: Marketing-oriented system using metric prefixes
- **Format**: Base-10 system (powers of 10)
- **Terms**: kilobyte, megabyte, gigabyte, terabyte, petabyte, exabyte, zettabyte, yottabyte
- **Origin**: Uses International System of Units (SI) metric prefixes
- **Usage**: Commonly used for marketing computer products to general public

### Binary Nomenclature
- **Purpose**: Engineering-oriented system using binary-based prefixes
- **Format**: Base-2 system (powers of 2)
- **Terms**: kibibyte, mebibyte, gibibyte, tebibyte, pebibyte, exbibyte, zebibyte, yobibyte
- **Origin**: Standard set by ISO/IEC, prefixes blend first letters of metric term with "binary"
- **Usage**: Used in computer engineering for numerical accuracy

## Storage Measurement Units

### Fundamental Units
- **Bit**: Binary digit storing either 1 (electric signal) or 0 (no signal)
- **Byte**: 8 bits representing a symbol or basic computer instruction

### Kilobyte (KB)
- **Decimal**: 10³ = 1,000 bytes
- **Binary (KiB)**: 2¹⁰ = 1,024 bytes
- **Decimal Inaccuracy**: -2.4% (-24 bytes)
- **Name Origin**: "Kilo" from Greek for "thousand"
- **Storage Capacity**: Short text file or small icon

### Megabyte (MB)
- **Decimal**: 10⁶ = 1,000,000 bytes
- **Binary (MiB)**: 2²⁰ = 1,048,576 bytes
- **Decimal Inaccuracy**: -4.9% (-48,576 bytes)
- **Name Origin**: "Mega" from Greek for "large"
- **Storage Capacity**: One minute of music or a short novel

### Gigabyte (GB)
- **Decimal**: 10⁹ = 1,000,000,000 bytes
- **Binary (GiB)**: 2³⁰ = 1,073,741,824 bytes
- **Decimal Inaccuracy**: -7.4% (-73,741,824 bytes)
- **Name Origin**: "Giga" from Greek for "giant"
- **Storage Capacity**: 2.5-3 hours of music or 300 high-resolution images

### Terabyte (TB)
- **Decimal**: 10¹² = 1,000,000,000,000 bytes
- **Binary (TiB)**: 2⁴⁰ = 1,099,511,627,776 bytes
- **Decimal Inaccuracy**: -10.0%
- **Name Origin**: "Tera" from Greek "tetra" (four) or "monster"
- **Storage Capacity**: 200,000 songs or 300 hours of video

### Petabyte (PB)
- **Decimal**: 10¹⁵ = 1,000,000,000,000,000 bytes
- **Binary (PiB)**: 2⁵⁰ = 1,125,899,906,842,624 bytes
- **Decimal Inaccuracy**: -12.6%
- **Name Origin**: "Peta" from Greek "penta" (five)
- **Storage Capacity**: 1.5 million CD-ROMs or 500 billion pages of text

### Exabyte (EB)
- **Decimal**: 10¹⁸ = 1,000,000,000,000,000,000 bytes
- **Binary (EiB)**: 2⁶⁰ = 1,152,921,504,606,846,976 bytes
- **Decimal Inaccuracy**: -15.3%
- **Name Origin**: "Exa" from Greek for "six"
- **Storage Capacity**: 11 million 4K movies or 3,000 copies of US Library of Congress

### Zettabyte (ZB)
- **Decimal**: 10²¹ = 1,000,000,000,000,000,000,000 bytes
- **Binary (ZiB)**: 2⁷⁰ = 1,180,591,620,717,411,303,424 bytes
- **Decimal Inaccuracy**: -18.1%
- **Name Origin**: "Zetta" from Latin "septem" (seven)
- **Storage Capacity**: 30 billion 4K movies

### Yottabyte (YB)
- **Decimal**: 10²⁴ = 1,000,000,000,000,000,000,000,000 bytes
- **Binary (YiB)**: 2⁸⁰ = 1,208,925,819,614,629,174,706,176 bytes
- **Decimal Inaccuracy**: -20.9%
- **Name Origin**: "Yotta" from Greek for "eight"
- **Storage Capacity**: Data from one million data centers
