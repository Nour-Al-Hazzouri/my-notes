# Week 1: Introduction to IT

## Introduction to IT Support
- **Information Technology**: The use of digital technology, like computers and the internet, to store and process data into useful information.
- **IT Specialist Responsibilities**: Managing, installing, maintaining, troubleshooting, and configuring office and computing equipment. Scope varies based on company size.

## History of Computing
- **Computer Definition**: A device that stores and processes data by performing calculations.
- **Early Computing**:
  - Punch cards were the first binary system
  - Algorithms are series of steps that solve specific problems
- **Development Milestones**:
  - Cryptography: The art of writing and solving codes
  - Enigma machine: German military invention cracked by Alan Turing's team
  - Magnetic tape: Stored data through magnetization
  - Software types: Commercial (paid) vs. open-source (freely distributed, modified, shared)

## Digital Logic
- **Binary System**: Base-2 numeral system (1s and 0s) used for computer communication
- **Bits and Bytes**:
  - Bit: A binary digit (0 or 1)
  - Byte: Group of 8 bits that can store one character (256 possible values)
- **Character Encoding**:
  - ASCII: Oldest standard, represented English alphabet using 127 values
  - UTF-8: Modern prevalent standard allowing variable byte usage
  - Unicode: Ensures consistent character representation
- **Color Representation**:
  - RGB model: Combines red, green, and blue values to create color range
  - Each pixel's color determined by RGB values represented in binary
- **Binary Functionality**:
  - Transistors control electrical voltage (1=voltage present, 0=no voltage)
  - Logic gates enable complex tasks by directing electrical signals based on conditions
- **Binary Counting**:
  - Position values: 128, 64, 32, 16, 8, 4, 2, 1 (total 255 plus zero)
  - Example: 0001010 = 8+2 = 10
  - ASCII example: Letter 'h' = 104 in decimal = 01101000 in binary (64+32+8)

## Computer Architecture Layers
- **Abstraction**: Simplifying complex systems by providing common interfaces
- **Four Main Layers**:
  1. Hardware: Physical components
  2. Operating System: Enables hardware-system communication
  3. Software: Human-computer interaction interface
  4. Users: Operate, maintain, and program the computer
