## Episode 1: Interface Introduction

### Setting Up Your Workspace
Make sure to have the menu set to "Essentials" to have the same workspace. To change it, go to Window → Workspace → Essentials.

### Interface Overview

#### Tools Panel
- Located on the far left
- Contains all usable tools in Illustrator
- Some tools work in sets (grouped together)

#### Color Controls
At the bottom of the Tools panel:
- White square: Fill color
- Black outline of a square: Stroke color

> 💡 **Tip:** To check a selected shape's colors, use the Selection Tool (`V`) and click on the shape. The color indicators will change to match the shape's colors.

#### Color Control Buttons
- **Swap button**: Swaps the fill and stroke colors
- **Default button**: Returns colors to original (black and white)
  - If a shape is selected, its colors will change too

#### Top Toolbars
- **Standard toolbar**: Contains various properties and controls
- **Control panel**: Shows properties specific to the currently selected tool

> 📝 **Note:** When you select a shape, the control panel changes accordingly. The fill and stroke colors can also be changed by clicking the arrow next to them.

#### Right Panel
A slim panel on the right side containing sub-panels represented by their icons.

#### Pasteboard
The gray area outside the artboard where you can place objects you don't want to include in the final artwork.

## Episode 3: Artboards

### Understanding Artboards
Artboards in Adobe Illustrator are canvases that make up a document.

#### Artboard Functions
- Multiple artboards can exist in a single document
- Useful for small document spreads like reflyers or flyers
- Can be used to create various versions of your artwork

#### Managing Artboards
If you have the example document open, you'll notice 6 artboards in the Artboard panel.

##### Navigating Artboards
- Double-click on an artboard to navigate to it
- Double-click the name to change it

##### Repositioning Artboards
1. Change an artboard's position
2. Refresh the artboards for the change to take effect
3. Click the button on the top right of the panel
4. Choose "Rearrange All Artboards"

> ⚠️ **Warning:** If an artboard is deleted, it must be removed from both the panel and the working area.

#### Creating New Artboards
From the Artboard panel:
1. Click the "+" icon after selecting the artboard you want it to be created after
2. Rearrange artboards as needed

Using the Artboard Tool:
1. Select the Artboard Tool from the Tools menu
2. Create an artboard using one of two methods:
   - Click and drag to make a custom size
   - Choose from the list of pre-sets

#### Modifying Artboards
To change an existing artboard's options:
1. From the Artboard menu, choose "Artboard Options" instead of rearranging
2. Make changes (e.g., switch from portrait to landscape)
3. For correct sizing, manually resize according to the provided dimensions

## Episode 4: Vector Basics

### Vector Fundamentals

#### Vector Properties
- Every shape made in Illustrator has a stroke outline
- A stroke that's joined creates a fill within
- Each shape is considered an individual element
- Elements can be selected, edited, and moved individually

#### Working with Layers
- Illustrator works with layers
- Multiple shapes can exist in one layer
- The bounding box color indicates which layer the shape is on
  - Example: If the box is red, the shape is on a red-colored layer

#### Shape Properties
With a shape selected, you can see its stroke and fill from the Tools menu.

#### Shape Manipulation
When a shape is selected:
- Eight white squares appear within the selection area
- These squares allow for simple modifications like resizing
- Position the mouse just outside the corners to rotate the shape

#### Transformation Tools
- **Scale Tool** (`S`): Reform and resize a selected shape
  - Hold `Shift` while dragging to maintain proportions
- **Rotate Tool** (`R`): Rotate the shape
  - Hold `Shift` to increment rotation in 45-degree steps

### Vector vs. Pixel Graphics

#### Direct Selection Tool
When using the Direct Selection Tool:
- Anchor points appear when a shape is selected
- A square shape will show 4 anchor points (black squares)

> 📝 **Note:** This is the key difference between Photoshop (pixel-based) and Illustrator (anchor-based).

#### Working with Anchor Points
- Anchor points contain mathematical values that create vector shapes
- Can be used for simple line strokes or complex vector shapes
- Fill color and stroke effects can be applied
- Vector artwork can be scaled to any size without losing quality

#### Manipulating Anchor Points
With the Direct Selection Tool:
1. Click on any anchor point to select it
2. Other anchor points will turn white, indicating they're not selected
3. Change the shape by moving the selected anchor point
4. Hold `Shift` + arrow keys to move using the keyboard
5. Select multiple anchor points by holding `Shift` + click (re-click to deselect)

#### Handles and Curves
- Vector parts with curves have anchor handles
- Handles determine curve values
- Each anchor point can have up to 2 handles
- Moving a handle changes the curve

## Episode 5: Fill & Stroke

### Vector Appearance

#### Basic Properties
- Vector shapes can have a fill color, a stroke color, or both
- A red dash indicates transparency or no color applied

#### Vector Structure
- Vectors are created from anchor points
- If vectors have curves, the anchors will have handles
- All anchors and handles together create a path and outline

#### Stroke Alignment
In Illustrator, strokes can align in three ways:
- Center
- Inside
- Outside

This can be changed from the Stroke menu.

#### Stroke Properties
From the Properties menu, you can modify:
- Corner styles
- Stroke types
- Stroke heads (can be changed to arrows)

## Episode 6: Using Color, Swatches, Pantones, Gradients & More

### Working with Swatches

#### Swatches Panel
- Contains pre-defined colors
- Includes effects like gradients and panel overlays
- Double-click a swatch to see more details about the color

#### Color Modes
A document's color mode can be:
- RGB (for digital work)
- CMYK (for printing)

### Color Selection Methods

#### Color Picker
Double-clicking the square from the Tools menu opens a color picker.

#### Saving Colors
- Save a color from the Tools menu by dragging it into the Swatches panel
- Store colors you create as color libraries

#### Color Panel
- Shows a complete color spectrum
- Cursor becomes an eyedropper when placed on it
- Selected color appears in the Tools menu

#### Color Guide Panel
Provides different shades of the color and other colors with similar tones.

#### Transparency Panel
Controls a shape's transparency.

### Working with Pantone Colors

#### Pantone vs. RGB
- RGB colors differ from Pantone colors
- Illustrator marks Pantone colors with white tabs that have spots

#### Accessing Pantone Colors
1. From the Swatches panel, click the top right menu
2. Choose Swatches Library → Color Books → Pantone + Solid Colors
3. If you know the number of your color, search for it in the search bar

### Sampling Colors
Use the Eyedropper Tool to pick colors from images:
1. Select an empty shape
2. Choose the Eyedropper Tool
3. Click on a color in the image to apply it to the shape

### Working with Gradients

#### Gradient Panel
- Requires a shape to be selected to become active
- Shows what colors make up the gradient and their distribution values

#### Creating Gradients
The easiest way to make a gradient:
1. Select the shape
2. Go to the Swatches panel and choose a pre-defined gradient
3. From the toolbar, choose the Gradient Tool (appears as a scroll bar)
4. Adjust the gradient as desired
5. Double-click on a color stop to change its color

#### Controlling Transparency
Use the Transparency panel to change the opacity of a shape.

## Episode 9: Grouping & Compounding Vectors

### Shape Stacking Order

#### Layer Order
- In Illustrator, newly created shapes appear at the bottom of the stack
- Each subsequent shape is placed on top
- Adjust order by right-clicking a shape and selecting Arrange → Bring to Front

### Working with Multiple Shapes

#### Individual vs. Group Editing
- When shapes are not grouped, they can be selected and edited individually
- When multiple shapes are selected, they will be edited as a group
- Hold `Shift` and click to select specific shapes

#### Grouping Shapes
- Press `Ctrl` + `G` to group multiple shapes into a single unit
- Edit individual shapes within a group by double-clicking on them
- Double-click on the canvas to return to group mode
- To ungroup, select the group and go to Object → Ungroup

### Combining Shapes

#### Using Pathfinder
- Turn a group of shapes into one shape using the Pathfinder panel
- Options include:
  - Unite: Joins shapes together
  - Exclude: When one shape is inside another, creates a hole

> ⚠️ **Note:** After using the Pathfinder tool, shapes can no longer be edited individually.

#### Shape Builder Tool
1. Select multiple overlapping shapes
2. Activate the Shape Builder Tool
3. Join shapes by dragging across them
4. Create new shapes from overlapping areas by clicking on them

## Episode 10: Drawing with the Pen Tool

### Pen Tool Basics

#### Getting Started
- The Pen Tool is used to create points by clicking
- Typically used with no fill (if fill color is active, it will fill the shape you draw)

#### Creating Straight Lines
- While using the Pen Tool, hold `Shift` to draw straight lines in specific directions
- Works in 4 main directions and 45-degree angles

#### Editing Paths
- After joining strokes, you'll have a vector shape
- Edit using the Direct Selection Tool
- For straight lines that can't be adjusted, add an anchor point with the Pen Tool
- Remove anchor points using the Anchor Point Remover Tool

### Best Practices

#### Minimizing Anchor Points
- Use as few anchor points as possible
- Requires judgment to anticipate where anchor points are needed for accurate curves

### Pen Tool Techniques

#### 1. Click and Drag Technique
- Click and drag to create anchor handles
- Create points at strategic locations
- Keep handles as straight as possible

#### 2. Click and Convert Technique
1. Select the Pen Tool and make straight lines
2. Create anchor points around the desired curve
3. Use the Anchor Point Tool to create handles
4. Hold `Shift` to make adjustments in 45-degree angles
5. Use the Direct Selection Tool for final adjustments

> 💡 **Tip:** `Shift` + `C` is the shortcut for the Anchor Point Tool.

#### Creating Curves
- Don't try to make a curve using just one anchor point
- Every anchor point works in harmony with others
- Another approach: make a straight line, then use the Anchor Point Tool to adjust it

### Alternative Drawing Tools

#### Pencil Tool
- Best for freehand strokes
- Creates more anchor points (more complex) since it's hand-drawn
- If your drawing isn't perfect, draw again over the problem area to refine it
- Use the Smooth Tool to make drawings smoother (select the line first)

> 💡 **Tip:** While drawing, press `A` for Direct Selection Tool, select the stroke, then press `N` to redraw and enhance.

#### Brush Tool
- Unlike the Pencil Tool, creates brush strokes rather than simple lines
- Can't redraw to enhance
- Uses brush effects from the Brush panel
- Brush effects can also be applied to pencil strokes

#### Choosing the Right Tool
- For clean geometric artwork: use the Pen Tool
- For hand-drawn artwork: use the Pencil Tool

## Episode 11: The Blob Brush Tool & Eraser Tool in Illustrator

### Blob Brush Tool

#### Basic Usage
- Press `Shift` + `B` to select the Blob Brush Tool
- Use `[` or `]` to change the size
- Allows drawing freehand strokes similar to the Brush Tool
- Unlike the Brush Tool, doesn't work along a vector path

#### Creating Vector Shapes
- Drawing with the Blob Tool creates a vector shape with no path inside
- Drawing on top of your first shape merges them together
- You can continuously build vector shapes this way

#### Working with Colors
- If scribbles made with the Blob Tool have different colors, they won't merge
- Only shapes with the same color will combine

## Episode 12: Type Tools in Adobe Illustrator

### Basic Text Creation
- After selecting the Type Tool, clicking once activates a typing object
- You can naturally change font and size

### Text Editing
- With the Selection Tool, double-tap into the typing element to make a selection in the text
- This allows for more specific edits

### Working with Paragraphs
- Click and drag the Type Tool to make an area for paragraph text
- If the area isn't big enough, a red box will appear
- Click on the red box and make another type area to continue the text
- The rest of the text will appear with a red line connecting the first typing area to the second

### Converting Text to Outlines
- Makes text into vector shapes
- Useful for certain types of edits
- Right-click on the text and choose "Convert to Outlines"

### Special Type Tools
- **Touch Type Tool** (`Shift` + `T`): Choose a specific letter and isolate it for individual editing
- **Vertical Type Tool**: Allows typing vertically

### Text in Shapes
- Convert a vector shape into a typing area using the Area Type Tool
- Select the vector shape with this tool
- If the shape isn't big enough:
  1. Use the Selection Tool to select the vector shape
  2. Use the Direct Selection Tool to click on the red box
  3. Click on the stroke of another vector shape to link them

### Text on Paths
- Use the Type on a Path Tool to create text that follows a path

## Episode 16: Compounding Vector Shapes & Strokes / Pathfinder Tool

### Understanding Compounding
In Illustrator, compounding means merging multiple vectors into a single vector.

> ⚠️ **Warning:** If you want to compound your vector shapes, always keep a backup of the original.

### Compounding Process

#### Steps to Compound Vectors
1. Go to Object → Expand Appearance
   - This will expand all brush strokes in the composition and turn them into vector shapes
2. Go back to the Object menu and press Expand
   - Choose options that suit your needs
3. Go to the Pathfinder menu and select Unite
4. From the menu button in the top right of the Pathfinder panel, choose Compound Shape
5. Click Expand

### Coloring Compounded Artwork

#### Steps for Coloring Artwork
1. Create a new artboard
2. Make a rectangle shape with a color you like, roughly the size of the new artboard
3. Make 2 copies of the shape
4. Set the first copy on the new artboard and align it perfectly
5. From the Pathfinder tool, select Exclude
6. Turn the color back to the color you chose
7. Move the second copy to fit the first one
8. Select them both with the artboard and then ungroup
9. Delete the artboard from the outside of the vector shape

## Episode 19: Prepare and Export Ready for Print in Adobe Illustrator

### Pre-Export Checklist

Before exporting, ensure you address these important points:

#### 1. Bleed Settings
- Most printers are okay with 5mm bleed
- May need adjustment in some cases
- Change from File → Document Setup menu
- Choose options that suit your needs
- Note that artboards may need to be rearranged after changing bleed settings

#### 2. Document Organization
- Each artwork should be on a separate layer to avoid problems later
- Properly name layers for easy identification
- Group related elements together

#### 3. Type Elements
- Consider converting type elements into vector shapes (creating outlines)
- This isn't required but can prevent font issues
- Right-click on text and choose "Create Outlines"

#### 4. Color Considerations
- For posters: Background color should remain
- For T-shirts: Artwork shouldn't have a background (as that will be the T-shirt color)
- Ensure only colors different from the T-shirt color are applied
- Verify colors are correct for the printer (Pantone colors if required)

#### 5. Export Settings
- Export each artwork as an individual file when possible
- When exporting, change the range to the number of artwork you want
- Choose `.PDF` format for print work

### PDF Export Options

#### Basic Settings
1. Choose Adobe PDF Preset: "High Quality Print" is recommended
2. From the Marks and Bleeds tab:
   - For posters: Check all printer's marks and use document bleed settings
   - For T-shirt printing: Don't check these options

> 📝 **Note:** The settings described above are primarily for posters. T-shirt printing may require different export settings.
