In the ES5 version of JS, local scoping was only possible inside of function, ES5 then introduced local scoping inside blocks of code using let and const

var can be used before it is declared, it can be redeclared and it’s scoped to a function or globally.

let & const can’t be used before declaration, can’t be redeclared, and are scoped to blocks
