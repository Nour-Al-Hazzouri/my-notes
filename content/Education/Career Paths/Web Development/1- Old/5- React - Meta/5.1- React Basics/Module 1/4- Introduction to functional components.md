- React components types:
	- Functional Components
	- Class Components
- Functional Components:
	- Every react app must contain at least one component, which is the root component that is loaded using the import statement
	- Render syntax is similar to self-closing tag in HTML
	- The roots components can  contain other components that developers create to represent the various UI part of the application
	  This component is ultimately converted to a DOM fragment and placed into the existing DOM as a child of the HTML div element with an ID of roots.
	  ![[Pasted image 20241118132902.png]]
	- React uses JSX- Combination of HTML and JS
	- React component won't render until it's used as a JSX element
	- When creating a component, it consists only of a JS file, and the name must be capitalized because react treast lowercase names as HTML elements
